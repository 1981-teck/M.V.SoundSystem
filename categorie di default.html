SKorganizer Pro - Categorie di Organizzazione Predefinite

SKorganizer Pro include un set di categorie predefinite per aiutarti a iniziare rapidamente con l'organizzazione dei tuoi file. Queste categorie coprono i tipi di file più comuni e associano estensioni specifiche a nomi di cartelle logici.

Puoi sempre personalizzare o aggiungere nuove regole di organizzazione in base alle tue esigenze specifiche nell'applicazione.

Elenco delle Categorie Predefinite

  • Immagini: .jpg, .jpeg, .png, .gif, .bmp, .tiff, .webp, .svg, .heic
  • Documenti: .pdf, .doc, .docx, .xls, .xlsx, .ppt, .pptx, .odt, .ods, .odp, .txt, .rtf, .md, .csv, .json, .xml
  • Musica: .mp3, .wav, .aac, .flac, .ogg, .wma, .m4a
  • Video: .mp4, .mov, .avi, .mkv, .flv, .wmv, .webm
  • Archivi: .zip, .rar, .7z, .tar, .gz, .bz2, .xz
  • Eseguibili: .exe, .msi, .dmg, .app, .bat, .sh, .run
  • Codice: .py, .java, .c, .cpp, .h, .html, .css, .js, .php, .rb, .go, .swift, .kt, .ts, .json, .xml
  • Fogli di Calcolo: .xls, .xlsx, .csv, .ods
  • Presentazioni: .ppt, .pptx, .odp
  • Font: .ttf, .otf, .woff, .woff2
  • Web: .html, .css, .js, .json, .xml, .php, .aspx, .asp
  • Dati: .json, .xml, .csv, .db, .sqlite, .sql
  • Backup: .bak, .tmp, .old, .zip, .rar
  • Log: .log
  • Torrenti: .torrent
  • ISO: .iso
  • Disegni Tecnici: .dwg, .dxf, .skp, .ai, .eps
  • Modelli 3D: .obj, .fbx, .stl, .blend
  • Audio Libri: .m4b, .aa, .aax
  • Note: .txt, .md, .rtf, .one
Nota: I file che non rientrano in nessuna delle categorie sopra elencate verranno spostati nella cartella predefinita "Altri File" se questa opzione è abilitata nelle impostazioni.
