SKorganizer Pro - Default Organization Categories

SKorganizer Pro includes a set of predefined categories to help you quickly get started with organizing your files. These categories cover the most common file types and associate specific extensions with logical folder names.

You can always customize or add new organization rules based on your specific needs within the application.

List of Default Categories

  • Images: .jpg, .jpeg, .png, .gif, .bmp, .tiff, .webp, .svg, .heic
  • Documents: .pdf, .doc, .docx, .xls, .xlsx, .ppt, .pptx, .odt, .ods, .odp, .txt, .rtf, .md, .csv, .json, .xml
  • Music: .mp3, .wav, .aac, .flac, .ogg, .wma, .m4a
  • Videos: .mp4, .mov, .avi, .mkv, .flv, .wmv, .webm
  • Archives: .zip, .rar, .7z, .tar, .gz, .bz2, .xz
  • Executables: .exe, .msi, .dmg, .app, .bat, .sh, .run
  • Code: .py, .java, .c, .cpp, .h, .html, .css, .js, .php, .rb, .go, .swift, .kt, .ts, .json, .xml
  • Spreadsheets: .xls, .xlsx, .csv, .ods
  • Presentations: .ppt, .pptx, .odp
  • Fonts: .ttf, .otf, .woff, .woff2
  • Web: .html, .css, .js, .json, .xml, .php, .aspx, .asp
  • Data: .json, .xml, .csv, .db, .sqlite, .sql
  • Backup: .bak, .tmp, .old, .zip, .rar
  • Logs: .log
  • Torrents: .torrent
  • ISO: .iso
  • Technical Drawings: .dwg, .dxf, .skp, .ai, .eps
  • 3D Models: .obj, .fbx, .stl, .blend
  • Audiobooks: .m4b, .aa, .aax
  • Notes: .txt, .md, .rtf, .one
Note: Files that do not fall into any of the above categories will be moved to the default "Other Files" folder if this option is enabled in the settings.
