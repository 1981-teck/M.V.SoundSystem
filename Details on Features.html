SKorganizer Pro v1.0: Your Solution for an Organized Computer

SKorganizer Pro is a powerful and intuitive application designed to help you keep your folders consistently organized. Forget clutter and let SKorganizer Pro automate the process of classifying and moving your files, freeing up space and enhancing your productivity.

Main Features

Intelligent File Organization

SKorganizer Pro offers various organization modes to suit your needs:

  • By Extension: Automatically classifies files into predefined categories such as Images, Documents, Music, Videos, Archives, Executables, and many others, based on their extension. You can rely on a comprehensive list of supported file types.
  • By Name Pattern: Define your custom rules using regular expressions to organize files based on specific words or patterns present in their names.
  • By Size: Create rules to move files to dedicated folders based on their size (e.g., small files, large files).
  • By Date: Organize your files into folders based on their modification date, choosing between automatic (YYYY-MM-DD) or custom formats.
  • Custom Rules: Manually map specific file extensions to custom destination folders, giving you complete control over organization.

Files that do not match any defined rule are automatically moved to an "Other Files" folder to keep everything tidy.

Automatic Folder Monitoring

Enable monitoring for your most used folders (like the Downloads folder). SKorganizer Pro will automatically detect new files or changes and organize them in real-time, without you having to do anything.

Advanced Duplicate Management

The application identifies and manages duplicate files to avoid wasting space. You can choose your preferred conflict resolution strategy:

  • Rename: Adds a suffix to the duplicate file to preserve it.
  • Overwrite: Replaces the existing file with the new one.
  • Ask: Allows you to decide on a case-by-case basis.
  • Skip: Does not move the duplicate file.

Advanced hashing algorithms (MD5, SHA1, SHA256, BLAKE2B) are used to ensure accurate duplicate identification.

Operation History and Undo Function

Every move operation is recorded, allowing you to view activity history. You have the option to undo one or more recent operations, either the last one or selected ones, and restore files to their original location or a designated recovery folder. History size is configurable to optimize memory usage.

Global Settings and Customization

SKorganizer Pro is highly customizable through a comprehensive settings panel:

  • Logging Level: Control the amount of information logged by the application.
  • Recycle Bin Management: Decide whether deleted files should go to the recycle bin or be permanently removed.
  • Maximum History Size: Set limits on the number of operations and the total size in MB of the history.
  • Hash Calculation: Configure the maximum file size for hash calculation and the timeout.
  • Custom Date Format: Define your own date formats for organization.
  • Application Theme: Choose your preferred graphical theme (System Default, Light, Dark, Blue, Green, Purple).

Intuitive User Interface (GUI)

The application features a user-friendly graphical interface based on PyQt5, making it simple to configure folders, manage rules, and monitor operation status.

Note on Performance: SKorganizer Pro has been designed with an emphasis on efficiency and memory optimization, utilizing advanced techniques such as LRU cache and optimized data structures to ensure a fluid experience even with a large volume of files. Please make configuration changes only if you are confident, and if necessary, test them on specially created folders. Please read the Instructions and EULA files.
