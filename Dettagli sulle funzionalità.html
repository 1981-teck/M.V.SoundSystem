SKorganizer Pro v1.0: La Tua Soluzione per un Computer Ordinato

SKorganizer Pro è un'applicazione potente e intuitiva progettata per aiutarti a mantenere le tue cartelle sempre organizzate. Dimentica il disordine e lascia che SKorganizer Pro automatizzi il processo di classificazione e spostamento dei tuoi file, liberando spazio e migliorando la tua produttività.

Caratteristiche Principali

Organizzazione Intelligente dei File

SKorganizer Pro offre diverse modalità di organizzazione per adattarsi alle tue esigenze:

  • Per Estensione: Classifica automaticamente i file in categorie predefinite come Immagini, Documenti, Musica, Video, Archivi, Eseguibili e molte altre, basandosi sulla loro estensione. Puoi contare su un elenco completo di tipi di file supportati.
  • Per Pattern di Nome: Definisci le tue regole personalizzate utilizzando espressioni regolari per organizzare i file in base a specifiche parole o schemi presenti nel loro nome.
  • Per Dimensione: Crea regole per spostare i file in cartelle dedicate in base alla loro dimensione (es. file piccoli, file grandi).
  • Per Data: Organizza i tuoi file in cartelle basate sulla loro data di modifica, scegliendo tra formati automatici (AAAA-MM-GG) o personalizzati.
  • Regole Personalizzate: Mappa estensioni di file specifiche a cartelle di destinazione personalizzate, dandoti il controllo completo sull'organizzazione.

I file che non corrispondono a nessuna regola definita vengono automaticamente spostati in una cartella per "Altri File" per mantenere tutto in ordine.

Monitoraggio Automatico delle Cartelle

Attiva il monitoraggio per le tue cartelle più usate (come la cartella Download). SKorganizer Pro rileverà automaticamente nuovi file o modifiche e li organizzerà in tempo reale, senza che tu debba fare nulla.

Gestione Avanzata dei Duplicati

L'applicazione identifica e gestisce i file duplicati per evitare sprechi di spazio. Puoi scegliere la strategia di risoluzione dei conflitti preferita:

  • Rinomina: Aggiunge un suffisso al file duplicato per conservarlo.
  • Sovrascrivi: Sostituisce il file esistente con il nuovo.
  • Chiedi: Ti permette di decidere caso per caso.
  • Salta: Non sposta il file duplicato.

Vengono utilizzati algoritmi di hashing avanzati (MD5, SHA1, SHA256, BLAKE2B) per garantire un'identificazione accurata dei duplicati.

Storico Operazioni e Funzione Annulla

Ogni operazione di spostamento viene registrata, permettendoti di visualizzare lo storico delle attività. Hai la possibilità di annullare una o più operazioni recenti, sia l'ultima che quelle selezionate, e di ripristinare i file nella loro posizione originale o in una cartella di recupero designata. La dimensione dello storico è configurabile per ottimizzare l'uso della memoria.

Impostazioni Globali e Personalizzazione

SKorganizer Pro è altamente personalizzabile tramite un pannello di impostazioni completo:

  • Livello di Logging: Controlla la quantità di informazioni registrate dall'applicazione.
  • Gestione Cestino: Decidi se i file eliminati devono andare nel cestino o essere rimossi permanentemente.
  • Dimensione Massima Storico: Imposta limiti al numero di operazioni e alla dimensione totale in MB dello storico.
  • Calcolo Hash: Configura la dimensione massima dei file per il calcolo dell'hash e il timeout.
  • Formato Data Personalizzato: Definisci i tuoi formati data per l'organizzazione.
  • Tema dell'Applicazione: Scegli il tema grafico che preferisci (Predefinito di Sistema, Chiaro, Scuro, Blu, Verde, Viola).

Interfaccia Utente Intuitiva (GUI)

L'applicazione dispone di un'interfaccia grafica user-friendly basata su PyQt5, che rende semplice configurare le cartelle, gestire le regole e monitorare lo stato delle operazioni.

Nota sulle Prestazioni: SKorganizer Pro è stato progettato con attenzione all'efficienza e all'ottimizzazione della memoria, utilizzando tecniche avanzate come cache LRU e strutture dati ottimizzate per garantire un'esperienza fluida anche con un grande volume di file, si prega di apportare modifiche alla configurazione solo se sicuri, eventualmente, effettuare test su cartelle create specificamente.
