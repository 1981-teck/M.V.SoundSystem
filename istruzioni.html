SKorganizer Pro - Guida Dettagliata all'Utilizzo

Benvenuto nella guida utente di SKorganizer Pro! Questo documento ti accompagnerà attraverso tutte le funzionalità dell'applicazione, aiutandoti a configurare e utilizzare al meglio le sue capacità di organizzazione dei file. Si consiglia di avviare SkOrganizer Pro in modalità amministratore per evitare problemi di permessi sui file.

Requisiti richiesti:

Sistema operativo: Windows 10, macOs (Darwin), Linux.

Hardware:

Minimo: Un processore dual-core moderno (ad esempio, Intel Core i3 o AMD Ryzen 3).

Consigliato: Un processore quad-core o superiore (Intel Core i5/i7 o AMD Ryzen 5/7) per prestazioni ottimali, specialmente durante l'organizzazione di un gran numero di file..

Memoria (RAM):

Minimo: 4 GB di RAM.

Consigliato: 8 GB di RAM o più per garantire un funzionamento fluido, soprattutto se si utilizzano contemporaneamente altre applicazioni o si gestiscono grandi quantità di dati.

Spazio di Archiviazione: Disco Rigido (HDD/SSD): Poco spazio richiesto per l'installazione del programma stesso (pochi MB). Tuttavia, è necessario spazio sufficiente sul disco rigido per i file che il programma dovrà organizzare. L'utilizzo di un'unità a stato solido (SSD) è fortemente consigliato per migliorare la velocità complessiva delle operazioni sui file.

Display: Risoluzione minima: 1024x768 pixel.

1. Iniziare

Al primo avvio, ti troverai di fronte all'interfaccia principale dell'applicazione, divisa in diverse schede per facilitare la navigazione e la configurazione.

Navigazione nell'Interfaccia

L'interfaccia utente è basata su Menù principali, che includeranno:

  • File-Gestisci Cartelle: Per aggiungere, configurare e gestire le unità o cartelle da organizzare.
  • File-Impostazioni globali: Per definire come i tuoi file devono essere classificati e spostati, cambiare il tema e i comportamenti dell'applicazione
  • Modifica: Per visualizzare le operazioni passate e annullarle.
  • Aiuto: Per visualizzare un riassunto delle funzionalità.
    • Attenzione: Non selezionare l'unità principale (es c:), comprometterebbe le funzionalità del sistema operativo se viene utilizzato il sistema "RICORSIVO" di ordinamento. IMPORTANTE: controllare che i file da organizzare non abbiano il "blocco" per file che provengono da un'altro computer, se sicuri della loro provenienza: tasto destro sul file, proprietà - togliere la spunta dal blocco, SkOrganizer Pro non può processare file con blocco di sicurezza, questo per proteggere il vostro computer.

2. Configurazione delle Cartelle

Questa è la prima fase per iniziare a organizzare. Devi indicare a SKorganizer Pro quali cartelle vuoi gestire.

Aggiungere e Rimuovere Cartelle

  1. Nella scheda Gestisci Cartelle, usa il pulsante "Aggiungi".
  2. Seleziona la cartella di origine che desideri organizzare o monitorare.
  3. Per ogni cartella aggiunta, potrai specificare se sarà destinata a Download (comportamento specifico) o Generale, dove i file verranno organizzati.
  4. Usa il pulsante "Impostazioni" per scegliere le varie impostazioni per quella cartella per esempio: Tipo cartella - Tipo di organizzazione (estensione, data, alfabetico, etc).
  • Usa il pulsante "Rimuovi Cartella" per eliminare una configurazione esistente.
    • Suggerimento: Inizia con una cartella "Downloads" o una cartella di prova per familiarizzare con il processo prima di applicarlo a directory più importanti. Consigliati backup su grosse operazioni ricorsive per nuove cartelle o unità con file già esistenti (e sottocartelle).

    3. Definizione delle Regole di Organizzazione

    Le regole determinano come SKorganizer Pro classificherà e sposterà i tuoi file.

    Tipi di Regole Disponibili:

    • Organizzazione per Estensione (Default):

      Questa è la modalità più semplice. SKorganizer Pro ha un elenco predefinito di categorie (es. Immagini, Documenti, Video) e le estensioni di file associate. I file verranno spostati nella cartella di destinazione appropriata basata sulla loro estensione.

      Esempio: Un file .jpg andrà nella cartella "Immagini".

    • Organizzazione Alfabetico:

      Organizza i file in base alla loro iniziale creando cartelle per ogni iniziale diversa presente.

      Esempio: I file Anna.txt Alberto.jpg verranno spostati in una cartella "A".

    • Organizzazione per Data:

      Organizza i file in base alla loro data (creazione/modifica) più vecchia. Puoi scegliere un formato di data automatico (AAAA-MM-GG) o definirne uno personalizzato.

      Esempio: I file creati/modificati il 15 marzo 2025 andranno in una cartella "2025-03-15".

    • Organizzazione per Dimensione:

      Definisci intervalli di dimensione (es. Minore di 1MB, Tra 1MB e 10MB, Maggiore di 100MB) e assegna una cartella di destinazione per ogni intervallo.

      Esempio: I file di grandi dimensioni (es. > 1GB) possono essere spostati in una cartella "Archivi Grandi".

    • Organizzazione per Pattern di Nome:

      Permette di definire espressioni regolari (regex) per abbinare i nomi dei file. I file che corrispondono al pattern verranno spostati in una cartella specificata.

      Esempio: Se crei un pattern per rapporto_.*\.pdf, tutti i PDF con nomi che iniziano con "rapporto_" andranno in una cartella "Rapporti annuali".

      Suggerimento: Le espressioni regolari sono potenti. Se non hai familiarità, cerca guide online o inizia con pattern semplici come .*nome_progetto.*.
    • Regole Personalizzate:

      Mappa manualmente specifiche estensioni di file a cartelle di destinazione personalizzate, indipendentemente dalle categorie predefinite.

      Esempio: Puoi decidere che tutti i file .log vadano nella cartella "Log del sistema".

    • Cartella "Altri File" (estensione) "#" (alfabetico):

      I file che non corrispondono a nessuna delle regole definite verranno spostati in una cartella designata per "Altri File" o "#" (configurabile nelle impostazioni).

    4. Esecuzione dell'Organizzazione

    Una volta configurate le cartelle e le regole, sei pronto per avviare il processo di organizzazione.

    Avvio Manuale

    Nella finestra principale seleziona una cartella col tasto destro poi seleziona "organizza ora", se vuoi organizzare tutte le cartelle nell'elenco seleziona "organizza tutti ora". Cliccando su questo, SKorganizer Pro inizierà a scansionare e spostare i file della cartella principale, (per le sottocartelle selezionare riorganizza ricorsivamente) in base alle tue regole.

    Monitoraggio Automatico

    Se hai abilitato il monitoraggio per una cartella tasto destro, o per utte le cartelle nella finestra principale Avvia Monitoraggio Tutti l'organizzazione avverrà automaticamente in background ogni volta che un nuovo file viene aggiunto o modificato in quella cartella.

    Stato e Notifiche

    L'applicazione fornirà aggiornamenti sullo stato dell'organizzazione nella finestra del log, mostrando i file spostati e le cartelle completate. In operazioni lunghe comparirà una barra di avanzamento.

    5. Gestione dei File Duplicati

    SKorganizer Pro è in grado di rilevare i file duplicati (stesso nome) e identici (stesso contenuto) utilizzando algoritmi di hashing per garantire accuratezza.

    Risoluzione dei Conflitti

    Quando viene rilevato un duplicato, l'applicazione agirà in base alla tua Politica di Risoluzione Conflitti, configurabile nelle impostazioni globali:

    • Rinomina: (default) Il file duplicato verrà rinominato (es. documento_copia_1.pdf) e spostato.
    • Sovrascrivi: Il file esistente nella destinazione verrà sostituito con il nuovo file. (Usa con cautela!)
    • Chiedi: L'applicazione ti chiederà come procedere per ogni duplicato rilevato. (non consigliabile su raccolte di file di "grosse dimensioni")
    • Salta: Il file duplicato non verrà spostato e rimarrà nella cartella di origine.
      • Attenzione: L'applicazione utilizza algoritmi di hashing crittografico (come MD5, SHA1, SHA256, BLAKE2B) per generare un'impronta digitale unica per il contenuto di ogni file. Se due file hanno lo stesso hash, SKorganizer Pro li considera IDENTICI nel loro contenuto, indipendentemente dal nome o dalla posizione e li sposta nel cestino, se sono invece DUPLICATI agirà secondo l'opzione selezionata (si raccomanda mantenere "default").

    6. Utilizzo dello Storico e Funzione Annulla

    SKorganizer Pro tiene traccia di tutte le operazioni di spostamento, offrendoti la possibilità di annullarle.

    Visualizzare lo Storico

    Nella scheda, Log Applicazione, potrai vedere un elenco di tutte le operazioni di organizzazione eseguite, inclusi i file spostati e le loro posizioni originali.

    Annullare Operazioni

    Nel Menù Modifica, puoi selezionare: Annulla Ultimo Spostamento o Annulla Operazioni per ripristinare i file nelle loro posizioni originali. L'applicazione usa il "Cestino di Sistema" per l'eliminazione dei file.

    Cartella Recupero

    Il programma possiede una cartella di recupero in cui salva i file in caso di errori o problemi di spostamento e annullamento operazione, questa si trova nell'unita principale del sistema, utenti/"nome utente"/Recupero.

    Suggerimento: Imposta un limite ragionevole per la dimensione dello storico nelle impostazioni globali per evitare un consumo eccessivo di memoria (modificare questa impostazione solo se esperti, se no lasciare quella di "default"). Attenzione: se il cestino di sistema ha l'opzione attiva "non spostare i file nel cestino. Rimuovi i file direttamente", questo influenzerà anche il programma, si consiglia di disattivarla.

    7. Impostazioni Globali

    Il menù File Impostazioni Globali ti permette di personalizzare il comportamento generale di SKorganizer Pro.

    • GENERALI:
    • Livello di Logging: Scegli il livello di dettaglio dei log dell'applicazione (utile per la risoluzione dei problemi).
    • Tema: Seleziona il colore dell'applicazione tra quelli che preferisci.
    • Max worker Threads: Definisce il numero massimo di operazioni che può eseguire contemporaneamente.
    • Max dimensione file per hash: definisce la dimensione massima di un file (in MB o GB) per cui l'applicazione calcolerà un valore hash per determinare se dei file sono identici.
    • Timeout hash: definisce il tempo massimo (in secondi) che l'applicazione attenderà per completare il calcolo dell'hash di un singolo file.
    • Suffisso esclusione: Indica il suffisso che devono avere i file per essere ignorati dal monitoraggio.
    • Cartella 'Altri File': Indica la cartella in cui verranno spostati i file non riconosciuti.
    • Estensioni Temporanee: Ti permette di indicare a SKorganizer Pro quali tipi di file considerare "di passaggio" o non importanti per l'organizzazione principale, consentendogli di gestirli in modo appropriato (solitamente ignorandoli o eliminandoli) per mantenere l'ordine..
    • Usa Cestino: Decidi se i file spostati o eliminati da SKorganizer Pro debbano finire nel cestino del sistema o essere rimossi permanentemente.
    • Dimensioni Storico: Imposta il numero massimo di operazioni da conservare nello storico e/o la dimensione massima in MB.
    • Formato Data Personalizzato: Se hai scelto l'organizzazione per data, puoi definire un formato di data specifico qui.
    • PULIZIA:
    • Dimensione Max File Pulizia: Indica la dimensione massima dei file presi in considerazione dalla funzione di Pulizia Automatica (tasto destro sulla cartella nella finestra principale).
    • Percorso Cestino: Seleziona percorso personalizzato.
    • Usa cestino per eliminazioni: Deselezione sconsigliata, nel caso, non utilizza il cestino.
    • COMPORTAMENTO:
    • Max Operazioni Annullabili: Personalizza quante operazioni annullabili vengono memorizzate (default 500, calcolato per un ottimo rapporto sicurezza-dimensioni).
    • Max dimensione cronologia: Personalizza quanto spazio massimo può utilizzare la cronologia.
    • Risoluzione conflitti: Seleziona cosa deve fare l'applicazione quando trova file duplicati.
    • Deduplica file: selezionare "Deduplica file" abilita un importante meccanismo di controllo che ti aiuta a mantenere l'integrità dei tuoi dati e a ottimizzare lo spazio su disco, mentre deselezionarla disattiva questo controllo, lasciando che i file vengano spostati senza verifiche sui duplicati di contenuto..
    • Algoritmo hash: Seleziona algoritmo personalizzato per verificare i file duplicati.

    8. Risoluzione dei Problemi e Consigli Utili

    • File Esclusi: I file con un suffisso specifico (es. .skorganizer_ignore) possono essere automaticamente esclusi dall'organizzazione. Controlla le impostazioni per questa opzione.
    • Politiche di Errore: SKorganizer Pro ha politiche interne per gestire gli errori di spostamento dei file (es. ritentare o spostare in una cartena di quarantena). Questo aiuta a prevenire la perdita di dati.
    • SkOrganizer è stato creato in modo di gestire i file, gli errori e tutti i processi con sistemi sofisticati prevenendo errori, usando il cestino di sistema per semplicità e usando sistemi di recupero e annullamento efficaci, a tal proposito si consiglia di non modificare le opzioni del programma se non si capiscono a fondo le meccaniche di tali modifiche o comunque di fare tali modifiche e testarle su cartelle create appositamente per evitare eventuali blocchi o perdite di dati.
    • Ultime raccomandazioni: si prega di leggere il file Dettagli sulle funzionalità e EULA.

    9. Funzione RICORSIVA

    • La funzione Ricorsiva serve per eseguire l'organizzazione su cartelle esistenti comprendendo anche le Sottocartelle o se si decide di cambiare il metodo di organizzazione, in questo caso apparirà una finestra di richiesta conferma all'utente, si prega di effettuare eventuali backup di sicurezza su file/cartelle importanti prima di eseguire questa azione, il programma potrebbe aver bisogno di "tempo" per esguire le operazioni su "grandissime" quantità di file o su computer poco potenti, interazioni errate, blackout, etc.. potrebbero causare perdite di dati SENZA possibilità di ripristino, TUTTI i file eliminati da questa funzione saranno nel "cestino di sistema" per offrire un ulteriore sistema di sicurezza.
